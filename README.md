LaTeX Template
==============

A small LaTeX template to get you started. Make sure to follow your
institution's style guides!

How to build
============

Use `make` on Linux, other systems, run these commands in exactly this order:

    pdflatex main.tex
    bibtex main        # NOTE: No file extension
    pdflatex main.tex
    pdflatex main.tex
